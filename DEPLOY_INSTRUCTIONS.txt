EVIE FARM - DEPLOYMENT INSTRUCTIONS
====================================

Since Vercel CLI is blocked, here's the easiest way to deploy:

METHOD 1: Deploy via Vercel Website (Recommended)
---------------------------------------------------
1. Go to: https://vercel.com
2. Sign in to your account
3. Click "Add New..." → "Project"
4. Click "Import Git Repository"
5. Find "evie-farm" (or connect GitHub if needed)
6. Click "Import"
7. Framework: Vite
8. Build Command: pnpm run build
9. Output Directory: dist
10. Click "Deploy"

Done! You'll get a URL like: https://evie-farm.vercel.app


METHOD 2: Deploy via Netlify (Alternative)
-------------------------------------------
1. Go to: https://app.netlify.com
2. Sign up/Login
3. Click "Add new site" → "Import an existing project"
4. Connect to GitHub
5. Select "evie-farm" repository
6. Build command: pnpm run build
7. Publish directory: dist
8. Click "Deploy"

Done! You'll get a URL like: https://evie-farm.netlify.app


WHAT YOU NEED TO DO FIRST:
---------------------------
You need to get the project files into your GitHub repository.

Since manual file copying is tedious, I'll create a simple package
you can download and extract into Desktop/evie-farm folder.

After that, use GitHub Desktop to:
1. Commit the changes
2. Push to GitHub
3. Then use Method 1 or 2 above to deploy
