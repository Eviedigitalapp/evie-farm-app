EASIEST WAY TO DEPLOY - STEP BY STEP
=====================================

Your repository is ready at: https://github.com/Eviedigitalapp/evie-farm-app

STEP 1: Upload Files to GitHub
-------------------------------
1. Go to: https://github.com/Eviedigitalapp/evie-farm-app
2. Click "uploading an existing file" link (or "Add file" → "Upload files")
3. Drag ALL these files/folders from your project:
   - src/ folder
   - public/ folder
   - supabase/ folder
   - utils/ folder
   - index.html
   - package.json
   - pnpm-lock.yaml
   - vite.config.ts
   - netlify.toml
   - .gitignore
   - All other files EXCEPT node_modules and dist folders

4. Scroll down and click "Commit changes"

STEP 2: Connect Netlify to GitHub
----------------------------------
1. Go to: https://app.netlify.com/
2. Click "Add new site" → "Import an existing project"
3. Click "Deploy with GitHub"
4. Authorize Netlify to access GitHub (if asked)
5. Search for and select: "evie-farm-app"
6. Build settings (should auto-detect):
   - Build command: pnpm run build
   - Publish directory: dist
7. Click "Deploy site"
8. Wait 2-3 minutes for build to complete

STEP 3: Deploy Supabase Edge Function
--------------------------------------
1. Go to Figma Make Settings Page
2. Click "Deploy Edge Functions" or "Deploy Server"
3. Wait for confirmation

DONE!
=====
Your app will be live at a Netlify URL!

Optional: Rename site in Netlify settings to get a better URL
